Game:
- Answer 15 questions, earn the money depending what level you finish on
- If you get a question wrong, you get sent back to the last checkpoint you passed

Features I have added:
- New theme and colours
- New layout of items such as timer and lifelines, better proportions on some of the objects
- New start screen
- Leaderboard integrated with an api to a database
- More info section
- Difficulty of questions to be used on the different tiers
- Checkpoints on every tier
- Playing alone and playing with audience
- The lifelines is now a popout menu
- Added all questions, from the word document Arhan provided (roughly 100 questions), and converted them into the required format
- Used AI to help categorise all questions in the file into easy medium hard and extreme difficulties
- Created shuffling mechanism, basically shuffles questions and gets a random order of questions for the game (previously the same questions would get repeated)
- More appealing certificate layout
- Deployed it to a web url so that it can be either redirected to from the website, or integrated in the website (using Netlify)
